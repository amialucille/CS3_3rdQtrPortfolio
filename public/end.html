THE END:

The End is an empty and silent dimension that consists of large floating islands that are above the void which is ruled by the ender dragon. This is the final dimension that the player goes through, being able tobe accessed by using an End Portal found in strongholds. In this dimension has the main boss of the the game and is the last requirement that needs to be done before finishing the game. There are no passive mobs that spawn in this dimension.

AREAS:

The End

The End is the massive central island of the dimension where the Ender Dragon resides. It contains tall obsidian towers with end crystals that regenerate the dragon’s health.

– POSSIBLE STRUCTURES • HOSTILE MOBS –

Possible Structures : Obsidian Platform, Obsidian Towers, Exit Portal, End Gateway

Hostile Mobs : Ender Dragon, Endermen

Small End Islands

Small End Islands are scattered floating islands found far from the central island. Chorus trees grow on these islands, which can be reached through End Gateways or by bridging long distances.

– POSSIBLE STRUCTURES • HOSTILE MOBS –

Possible Structures : End Gateway Portals

Hostile Mobs : Endermen

End Midlands

End Midlands are part of the outer End islands and do not generate chorus trees. These areas are usually at a lower elevation compared to nearby islands.

– POSSIBLE STRUCTURES • HOSTILE MOBS –

Possible Structures : End Cities, End Gateways, End Ships

Hostile Mobs : Endermen

End Highlands

End Highlands are elevated hilltop areas of the outer End islands. These biomes often have the highest terrain and are where End Cities generate.

– POSSIBLE STRUCTURES • HOSTILE MOBS –

Possible Structures : End Cities, Chorus Trees, End Gateways, End Ships

Hostile Mobs : Shulkers, Endermen

End Barrens

End Barrens generate along the outer edges of the End Midlands, forming cliffs and steep ledges with little terrain variety.

– POSSIBLE STRUCTURES • HOSTILE MOBS –

Possible Structures : End Gateways

Hostile Mobs : Endermen
